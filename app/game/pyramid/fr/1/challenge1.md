# Un message texte mystérieux

Lorsque vous remettez votre guide dans votre sac à dos, votre téléphone portable commence à bourdonner. C'est étrange, car vous êtes loin de tout signal. Vous ouvrez votre téléphone, mais votre application de SMS se bloque.

Pour lire vos textes, vous devrez les visualiser sur le web.

Clonez [ce dépôt](https://github.com/MicrosoftDocs/Azure-Maya-Mystery-Challenge-1) et suivez les instructions sur le fichier README pour lancer le site sur Azure Static Apps.
